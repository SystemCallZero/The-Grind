# The Grind

Приложение для отслеживания тренировок или иных задач.

## Возможности
- Создание дней тренировок
- Добавление упражнений
- Отметка выполненных упражнений
- Прогресс-бар и система уровней
- Автосохранение в JSON

## Сборка
Проект написан на C++ с использованием Qt 6.11.0.
